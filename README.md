# SpartanHub

**SpartanHub** is a faculty-focused knowledge base for San Jos√© State University that centralizes teaching resources, supports AI-assisted discovery, and enables collaboration through curated content and contextual chat.

The platform allows administrators to upload and manage academic resources while enabling faculty to browse, search, and interact with content using an AI assistant powered by Google Gemini.

---

## Features

- Centralized knowledge base for teaching & learning materials  
- Filterable dashboard (subject, level, resource type)  
- Related content linking (bidirectional)  
- Admin console for uploading & managing resources  
- AI chat assistant with contextual awareness of selected resources  
- File uploads (PDF, DOCX, PPTX, video, images, spreadsheets, code)  
- External link support (Google Docs, Slides, YouTube, etc.)  
- Role-based access control via Supabase Auth  
- Deployed with Vercel (frontend) and Render (backend)
